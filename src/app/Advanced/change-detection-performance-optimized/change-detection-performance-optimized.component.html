Optimierte Performance

Durch ChangeDetectionStrategy.OnPush wird die ChangeDetection nur ausgelöst, wenn die Input Variablen mit einem neuen Objekt beschrieben werden. Durch immutable Objekte werden die Daten (hier die ContactList) also nicht direkt verändert (durch die OnPush Strategie würde die UI dann nicht automatisch aktualisieren). Stattdessen wird das ContactList Objekt mit einem neuen Objekt beschrieben (neu "gepusht").
