Wrapper Direktiven (JQuery Slider)

Mit Direktiven können externe Bibliotheken, welche keine Angular Integration haben gewrappt werden. Damit kann die Direktive als Wrapper für die Bibliothek in Angular verwendet werden. So kann z.B. das two-way-property Binding mit dem Jquery Slider kombiniert werden.
